# Fullstack Web API Project

A React-based web application that fetches book data from an API and allows users to browse books, add them to a cart, and manage interactions through a clean user interface.

## Features
- Fetch books from API
- Add books to cart
- Dynamic UI with React
- Responsive design

## Technologies Used
- React
- JavaScript
- HTML
- CSS
- REST API
